# FinTrack

App de controle financeiro descomplicado

## Proposta

A plataforma FinTrack é sistema de acompanhamento financeiro pessoal, utilizando informações de movimentação monetária própria, o sistema promete te oferecer um feedback visual mais amigável e simples do que uma tabela complexa cheia de células e alterações confusas.

Dentro da plataforma, o fluxo é simples, crie sua conta, preencha as informações básicas como salário e entradas extras(freela, bico ou mesada), despesas(contas, lanches, gasolina) e seus investimentos(CDB, FII, Tesouro), o simples que funciona, assim obtendo gráficos e a diferença entre seus ganhos e suas "perdas".
